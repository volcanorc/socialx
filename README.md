# SocialX

SocialX keeps all my accounts in one place.

I use it to organize my real accounts and dummy accounts so I can stop forgetting them and find everything faster.

## What it is for

- Keep all my accounts in one place
- Remember usernames, emails, notes, and links between accounts
- Search and sort accounts quickly
- Keep my account list simple and easy to manage

## Simple use

- Sign in
- Add an account
- Search or filter when I need it
- Edit or update anything later

SocialX is a personal account organizer made to keep everything in one clean place.
